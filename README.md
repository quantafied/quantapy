# QuantaPy

Python library for quantitative trading research, strategy development, and backtesting with an emphasis on developing robust models considering uncertainty. 

# Motivation

Optimizing model parameters by backtesting historical data suffers from overfitting. This is a single observation sampled from an underlying distribution of a stochastic process. Backtests are simulations with assumptions of ideal conditions which do not exist in reality. Performant models that can survive the uncertainty of live trading environments need to be designed with these limitations in mind. 

This library is under active development and does not contain a stable release yet
